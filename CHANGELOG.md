# Changelog

숏폼 연구소 · 템플릿 저장공간의 변경 이력. [Semantic Versioning](https://semver.org/lang/ko/)을 따르며, 과거는 지워지지 않고 아래에 **누적**됩니다.

---

## Unreleased

### Changed
- **사이드바 하위 메뉴 여백 개선.** `nav-child`의 최소 높이와 내부 패딩을 키우고 항목 사이 간격을 추가해 hover·클릭 영역과 가독성을 개선.
- **홈 템플릿 임베드 목록.** 기존 `두 갈래로 시작하세요` 섹션을 제거하고 그 자리에 기준 영상 미리보기·템플릿명·플랫폼 배지를 갖춘 가로형 카드 목록을 배치. 모바일에서는 가로 스크롤로 탐색하도록 구성.
- **홈 히어로 통계 제거.** 연구 매체·분석 연구일지·분석 장면 숫자 블록을 삭제해 핵심 소개와 행동 유도에 집중.
- **브랜드명 통일.** 사이드바 브랜드, 브레드크럼, 동적 문서 제목을 `숏폼광고 영상제작 연구소`로 변경.

## v1.7.1 — 2026.07.13

### Removed
- **사이드바 브랜드 마크.** 사이드바 헤더의 프로필 이미지와 전용 스타일, 사용하지 않는 SVG 자산을 제거.
- **파비콘.** 문서 `<head>`에 포함되어 있던 인라인 SVG 파비콘 선언을 제거.

## v1.7.0 — 2026.07.11

### Added
- **영상별 맞춤 프롬프트 제작 가이드.** 질문 목록이 `templates.json`의 `builder.fields`로 영상 유형마다 다르게 정의됨. 질문형 라벨 + 텍스트필드, 연령대는 중복선택 가능한 칩 버튼, 말투는 단일선택 칩. [프롬프트 제작하기] 버튼을 누르면 복사해서 AI에 바로 전달 가능한 상세 프롬프트가 생성됨(생성 후에는 입력 변경이 실시간 반영).
- **영상 제작의 흐름.** profile-dossier에 기준 영상의 단계별 제작 순서(PainPoint → 타겟 공감 → … → GainPoint + 공감)를 표시. `flow` 필드가 정본.
- **판매상품·제작 난이도.** 레퍼런스·제작 자료 항목을 대체.

### Changed
- **홈 화면 전면 개편.** '숏폼광고 영상제작 연구소' — 매체별·상품별·유형별 VA 분석 연구일지, KPI·ROAS 연구 기반, 복제가 아닌 새로운 VA 패러다임 형성이라는 정체성으로 콘텐츠 재작성.
- **디자인 총점검(뷰티 파스텔).** 보라 팔레트를 부드러운 로즈 파스텔로 교체. 폰 목업·radial 블롭·거대 워터마크 숫자·영문 장식 라벨 등 AI 티 나는 요소 제거, trend-detail-hero 라이트 카드화, 버튼·칩·카드를 토스류의 절제된 스타일로 정돈.
- '이 영상 프로필이 필요한 순간'(bestFor) 섹션 제거(데이터는 유지).

## v1.6.0 — 2026.07.11

### Added
- **영상 분석 리포트.** 템플릿 상세에 힉스필드 장면 분석 기반의 실제 분석 내용을 다섯 항목(후킹 / 구성·전개 / 카피·스크립트 / 편집·비주얼 리듬 / 전체 품질 분석)으로 구분해 표시. `templates.json`의 `report` 필드가 정본.
- **프롬프트 제작 가이드.** 변수 입력 필드(제품·타깃·문제·효익·근거·말투·필수 정보·CTA·길이)를 배치하고, 입력값이 최종 프롬프트에 실시간 반영되어 자동 작성. 비워 둔 변수는 AI가 '제안'으로 채우도록 안내.

### Changed
- **용어 정책: '해부' 금지.** 전 화면·데이터·문서에서 '해부'를 '분석'으로 치환(템플릿 제목 포함). `validate.mjs`가 위반을 검증.
- **`.eyebrow` 클래스 전면 제거.** 마크업·스타일 모두 삭제.
- **사이드바 nav-child 표기.** 템플릿 제목 대신 동영상 유형의 짧은 표현(`navLabel`, 예: 블러셔 문제제기 반전)을 표시.

## v1.5.0 — 2026.07.11

### Added
- **새 브랜드 프로필 이미지.** 사용자가 제공한 Microsoft Fluent 비디오 카메라 SVG를 라이선스 주석과 함께 로컬 자산으로 편입.
- **단일 완성형 재현 블루프린트.** 후킹, 구성·전개, 카피·스크립트, 편집·비주얼 리듬, 장면표, 촬영 대안, 품질 검수를 한 번에 생성하는 제작 프롬프트로 상세 화면 개편.

### Changed
- **AI 안티디자인 타이포그래피 패치.** 표시 글자 최소 13px, 자간 `-0.01em~0.01em`, Pretendard 단일 폰트 규칙을 전역 적용.
- **시맨틱 마크업 정리.** 장식용 `<i>` 태그를 의미가 드러나는 `<span>` 요소로 교체하고, 제목 줄바꿈·터치 영역·버튼 눌림 피드백을 보완.
- **영상 상세 가독성 개선.** `profile-dossier`의 레이블·본문·목록 크기와 행간을 높이고 불필요한 영문 장식 레이블을 한국어 문맥 레이블로 교체.
- **분석 프롬프트 UI 제거.** 영상 분석 노트의 후킹·구성·카피·편집 개별 프롬프트를 제거하고 재현 블루프린트만 노출.

## v1.3.0 — 2026.07.11

### Added
- **재현 블루프린트 5종.** 기준 영상 5개를 힉스필드 MCP로 전부 장면 분석(25+13+22+12+9 = 총 81장면)하고, 각 분석에서 "복사 붙여넣기 한 번 + [교체 변수]만 수정하면 캡컷·캔바 등 어떤 툴에서도 같은 느낌으로 재현 가능한" 제작 명세(`blueprint`)를 도출해 템플릿마다 삽입. 카드에 블루프린트 섹션(복사 버튼 + 아코디언) 신설.
- **블루프린트 규격 문서화.** [docs/video-analysis.md](docs/video-analysis.md)에 "재현 블루프린트" 절 추가 — 4섹션 규격(사용법/교체 변수/장면 구성/촬영·편집 규칙)과 작성 원칙. 템플릿 제작 워크플로(CLAUDE.md)에 분석→블루프린트 단계 편입. `validate.mjs`가 규격 준수를 검증.

### Changed
- **기준 영상 실내용 반영.** 분석 결과에 맞춰 5개 템플릿의 `reference.title/why` 갱신. 블러셔 브러쉬 영상(교과서적 UGC 리뷰)을 리뷰·UGC 광고형으로, BB크림 영상을 밈·트렌드형 슬롯으로 상호 스왑. ※ 밈·트렌드형의 기준 영상은 리뷰형이라 아키타입과 성격이 다름 — 추후 트렌드 영상으로 교체 권장(카드에 표기).

## v1.0.0 — 2026.07.11

### 전환 (Breaking)
- **숏폼 연구소로 전면 전환.** 기존 **UIUX-DH · Unified Design System**을 대체. 컴포넌트(25종)·토큰·데모(18종)·릴리스 등 이전 디자인시스템 자산과 관련 디렉토리(`components/` · `tokens/` · `docs/` · `foundations/` · `schemas/` · `snippets/`)를 모두 제거했습니다.

### Added
- **템플릿 저장공간.** 기준 숏폼 영상을 항목별로 분석하는 분석 프롬프트 아카이브. 각 템플릿은 기준 영상(`reference`) · 사용성(`bestFor`) · 장단점(`pros`/`cons`) · 항목별 프롬프트(`prompts`)로 구성.
- **분석 프롬프트 5항목 규격:** 후킹(`hook`) · 구성·전개(`structure`) · 카피·스크립트(`script`) · 편집·비주얼 리듬(`edit`) · 내 주제로 재현(`remix`).
- **시드 템플릿 5종:** 후킹 폭발형 · 스토리텔링형 · 정보/튜토리얼형 · 리뷰/UGC 광고형 · 밈/트렌드·챌린지형. (총 25개 프롬프트)
- **복사 중심 UX:** 프롬프트별 복사 + 템플릿 전체 프롬프트 복사, 성공 토스트. `file://` 대비 clipboard 폴백 포함.
- **탐색:** 아키타입 필터 칩(사이드바·상단) + 키워드 검색, 사용성/장단점/기준영상 표시.
- **`templates.json` 단일 진본:** 데이터 추가 시 화면·사이드바·필터가 자동 갱신.
- **도구:** `scripts/serve.mjs`(의존성 0 로컬 서버) · `scripts/validate.mjs`(무결성 검증).

### Kept
- 라이트/다크 테마 토큰 체계(`--sm-*` / `--p-*`)와 Pretendard 타이포그래피 기반은 계승해 새 UI에 재사용.

### Removed
- `index.html`의 디자인시스템 마크업 전체, `system.json`, `handoff.md`, `assets/css/_tokens.generated.css`, 토큰 빌드·문서 생성 스크립트(`build-tokens.mjs` · `gen-docs.mjs`).
