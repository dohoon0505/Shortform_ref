# Changelog

숏폼 연구소 · 템플릿 저장공간의 변경 이력. [Semantic Versioning](https://semver.org/lang/ko/)을 따르며, 과거는 지워지지 않고 아래에 **누적**됩니다.

---

## v1.5.0 — 2026.07.11

### Added
- **새 브랜드 프로필 이미지.** 사용자가 제공한 Microsoft Fluent 비디오 카메라 SVG를 라이선스 주석과 함께 로컬 자산으로 편입.
- **단일 완성형 재현 블루프린트.** 후킹, 구성·전개, 카피·스크립트, 편집·비주얼 리듬, 장면표, 촬영 대안, 품질 검수를 한 번에 생성하는 제작 프롬프트로 상세 화면 개편.

### Changed
- **AI 안티디자인 타이포그래피 패치.** 표시 글자 최소 13px, 자간 `-0.01em~0.01em`, Pretendard 단일 폰트 규칙을 전역 적용.
- **시맨틱 마크업 정리.** 장식용 `<i>` 태그를 의미가 드러나는 `<span>` 요소로 교체하고, 제목 줄바꿈·터치 영역·버튼 눌림 피드백을 보완.
- **영상 상세 가독성 개선.** `profile-dossier`의 레이블·본문·목록 크기와 행간을 높이고 불필요한 영문 장식 레이블을 한국어 문맥 레이블로 교체.
- **분석 프롬프트 UI 제거.** 영상 해부 노트의 후킹·구성·카피·편집 개별 프롬프트를 제거하고 재현 블루프린트만 노출.

## v1.4.0 — 2026.07.11

### Added
- **플랫폼별 트렌드 문서.** YouTube Shorts·Instagram Reels·TikTok의 플랫폼 성격, 핵심 시청 행동, 효과적인 광고 영상 공식과 제작 체크리스트를 공식 자료 기반으로 추가.
- **템플릿 개별 상세 페이지.** 목록에서 템플릿을 고르면 큰 9:16 기준 영상과 영상 프로필, 분석 프롬프트, 재현 블루프린트를 별도 화면에서 확인하도록 분리.

### Changed
- **문서형 사이드바 재구성.** 홈·트렌드·템플릿·사용법 계층과 하위 문서 링크를 한눈에 탐색하도록 개편.
- **사용자 화면 정리.** 템플릿 내부 ID, 데이터 파일명, GitHub 링크, 개발자용 추가·변경 이력 화면을 제거하고 제작에 필요한 정보만 표시.

## v1.3.0 — 2026.07.11

### Added
- **재현 블루프린트 5종.** 기준 영상 5개를 힉스필드 MCP로 전부 장면 분석(25+13+22+12+9 = 총 81장면)하고, 각 분석에서 "복사 붙여넣기 한 번 + [교체 변수]만 수정하면 캡컷·캔바 등 어떤 툴에서도 같은 느낌으로 재현 가능한" 제작 명세(`blueprint`)를 도출해 템플릿마다 삽입. 카드에 블루프린트 섹션(복사 버튼 + 아코디언) 신설.
- **블루프린트 규격 문서화.** [docs/video-analysis.md](docs/video-analysis.md)에 "재현 블루프린트" 절 추가 — 4섹션 규격(사용법/교체 변수/장면 구성/촬영·편집 규칙)과 작성 원칙. 템플릿 제작 워크플로(CLAUDE.md)에 분석→블루프린트 단계 편입. `validate.mjs`가 규격 준수를 검증.

### Changed
- **기준 영상 실내용 반영.** 분석 결과에 맞춰 5개 템플릿의 `reference.title/why` 갱신. 블러셔 브러쉬 영상(교과서적 UGC 리뷰)을 리뷰·UGC 광고형으로, BB크림 영상을 밈·트렌드형 슬롯으로 상호 스왑. ※ 밈·트렌드형의 기준 영상은 리뷰형이라 아키타입과 성격이 다름 — 추후 트렌드 영상으로 교체 권장(카드에 표기).

## v1.2.0 — 2026.07.11

### Changed
- **타이포그래피 스케일 확대.** 가독성 개선을 위해 전 텍스트 토큰을 한 단계 키움(본문 15→16px, 라벨 13→14px, 헤딩·디스플레이 +2~4px, 프롬프트 본문 13→14.5px). 사이드바 폭 288→300px.
- **브레드크럼 뎁스 표시.** 템플릿 저장공간에서 스크롤 위치의 템플릿까지 표시 — 예: `숏폼 연구소 · 템플릿 저장공간 · 후킹 폭발형 해부` (스크롤스파이).
- **동영상형 브랜드 프로필.** '숏' 텍스트 마크를 세로(9:16) 영상 프레임 + 재생 버튼 SVG 로고(인디고 그라데이션 타일)로 교체. 같은 모양의 SVG 파비콘 추가.

## v1.1.0 — 2026.07.11

### Added
- **기준 영상 임베드.** 영상을 저장소에 저장하지 않고 `reference.url`로 임베드(`.mp4` → `<video>` 네이티브 플레이어, 그 외 URL → `<iframe>`). 카드에서 바로 재생 가능.
- **시드 템플릿 5종에 기준 영상 연결.** Glovv 레퍼런스 컬렉션 CDN 영상 5개를 템플릿 순서대로 `reference.url`에 매핑.
- **영상 분석 방법 문서 [docs/video-analysis.md](docs/video-analysis.md).** 영상 분석은 항상 힉스필드(Higgsfield) MCP 사용 — `media_import_url` → `video_analysis_create` → `video_analysis_status` 폴링 절차, 50MB 제한·폴링 간격·기존 분석 재사용 등 주의사항, 분석 결과(scenes)를 템플릿 프롬프트와 결합하는 방법.

### Changed
- CLAUDE.md · AGENTS.md에 영상 분석 규칙(힉스필드 MCP 필수, 문서 필독)과 임베드 규칙 반영.

## v1.0.0 — 2026.07.11

### 전환 (Breaking)
- **숏폼 연구소로 전면 전환.** 기존 **UIUX-DH · Unified Design System**을 대체. 컴포넌트(25종)·토큰·데모(18종)·릴리스 등 이전 디자인시스템 자산과 관련 디렉토리(`components/` · `tokens/` · `docs/` · `foundations/` · `schemas/` · `snippets/`)를 모두 제거했습니다.

### Added
- **템플릿 저장공간.** 기준 숏폼 영상을 항목별로 해부하는 분석 프롬프트 아카이브. 각 템플릿은 기준 영상(`reference`) · 사용성(`bestFor`) · 장단점(`pros`/`cons`) · 항목별 프롬프트(`prompts`)로 구성.
- **분석 프롬프트 5항목 규격:** 후킹(`hook`) · 구성·전개(`structure`) · 카피·스크립트(`script`) · 편집·비주얼 리듬(`edit`) · 내 주제로 재현(`remix`).
- **시드 템플릿 5종:** 후킹 폭발형 · 스토리텔링형 · 정보/튜토리얼형 · 리뷰/UGC 광고형 · 밈/트렌드·챌린지형. (총 25개 프롬프트)
- **복사 중심 UX:** 프롬프트별 복사 + 템플릿 전체 프롬프트 복사, 성공 토스트. `file://` 대비 clipboard 폴백 포함.
- **탐색:** 아키타입 필터 칩(사이드바·상단) + 키워드 검색, 사용성/장단점/기준영상 표시.
- **`templates.json` 단일 진본:** 데이터 추가 시 화면·사이드바·필터가 자동 갱신.
- **도구:** `scripts/serve.mjs`(의존성 0 로컬 서버) · `scripts/validate.mjs`(무결성 검증).

### Kept
- 라이트/다크 테마 토큰 체계(`--sm-*` / `--p-*`)와 Pretendard 타이포그래피 기반은 계승해 새 UI에 재사용.

### Removed
- `index.html`의 디자인시스템 마크업 전체, `system.json`, `handoff.md`, `assets/css/_tokens.generated.css`, 토큰 빌드·문서 생성 스크립트(`build-tokens.mjs` · `gen-docs.mjs`).
